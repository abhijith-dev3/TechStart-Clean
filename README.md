# TechStart-Clean

## Live Demo
[View the site](https://abhijith-dev3.github.io/TechStart-Clean/)

## Overview
TechStart-Clean is a simple, responsive landing page built with HTML and CSS.  
It demonstrates clean design, a navigation bar, hero section, features, pricing, contact form, and footer.

## Features
- Fully responsive layout
- Hero section with image and call-to-action
- Features section with grid layout
- Pricing plans section
- Contact form
- Clean, modern design

## Tech Stack
- HTML5
- CSS3
